# 录屏工具

屏幕录像保存至 webm 文件
可选择已打的窗口录像
可自定义 utools 全局快捷键关键字“开始录屏”“停止录屏”
可使用 chrome 或 VLC 等主流播放器播放 webm
视频编码 h264
MacOS Linux 无法内录音频

---

MacOS 和 Linux 无法录制声音
