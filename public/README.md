# 录屏工具

屏幕录像保存至 webm 文件
可选择已打开的窗口录像
可自定义 utools 全局快捷键关键字“开始录屏”“停止录屏”“暂停恢复录像”
可使用 chrome 或 VLC 等主流播放器播放 webm
视频编码 h264 可使用另一个"FFmpeg 助手"插件转换格式

---

MacOS Linux 无法内录音频 可使用 soundFlower 或 blackHole 等虚拟声卡软件转录到 mic
